# EscapeWinterHouseTutorial
This is my Unity project tutorial, the story is about the winter house murder. And the game is a single-player puzzle game, which is for my course project.

## 製作遊戲步驟
- 目錄
 1. 製作美術與UI圖形
 2. 將介面拖入Unity布局
 3. 設定功能
 4. Unity C# Script撰寫
 5. 測試是否能跑動與打包
