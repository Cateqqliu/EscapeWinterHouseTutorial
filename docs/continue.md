# 🚀 寫在最後：未完待續的《Winter House》

到目前為止，我們已經成功建構了《Winter House》的基礎骨架。玩家可以進入場景、點擊證物、聆聽環境音，並順利迎來結局。

但身為一名專業的調查員，你可能會發現這棟冬日別墅裡，似乎還有更多未解的機關與深不可測的秘密等待挖掘。沒錯，為了讓教學節奏保持清晰，我們將許多「進階的密室逃脫機制」保留到了未來。

以下是我們即將在未來解鎖的進階開發清單：

### 🎒 1. 證物背包與圖鑑收集系統
* **進度預告：** 點擊道具後，除了彈出對話框，道具將會在場景中「消失」，並自動收納進玩家的 UI 背包中，右上角的圖鑑數字也會隨之 `+1`。

### 🔐 2. 複合機關與密碼鎖設定 (Input Field)
* **進度預告：** 只有找到正確的線索，玩家才能在 UI 面板的輸入框中打上正確的密碼，解開隱藏在書房裡的保險箱。

### 🚪 3. 多房間探索與狀態保留
* **進度預告：** 遊戲將不再侷限於單一的 Play 場景。我們將實作「客廳 ➔ 廚房 ➔ 地下室」的無縫切換，並確保玩家在不同房間穿梭時，收集到的證物與打開的門鎖狀態不會被重置。
